SSL/TLS-Verschlüsselung

Zu Sicherheitszwecken verwendet unsere Website SSL/TLS-Verschlüsselung, die von GitHub bereitgestellt wird, um die Übertragung sensibler Inhalte wie der über Kontaktformulare übermittelten Informationen zu schützen. Eine verschlüsselte Verbindung ist am "https://" Adresspräfix und am Schlosssymbol in der Adressleiste Ihres Browsers erkennbar. Diese Funktion stellt sicher, dass an uns übertragene Daten nicht von Dritten gelesen werden können.

Bitte beachten Sie, dass die SSL/TLS-Verschlüsselung von GitHub verwaltet wird und wir keinen direkten Einfluss auf diesen Aspekt der Sicherheit unserer Website haben.
